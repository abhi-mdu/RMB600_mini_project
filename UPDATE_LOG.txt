## Technical Details

### MoveJ Implementation
- **Function**: `MoveJ(T_start, T_end, robot, toolFrame)`
- **Type**: Joint-space motion planning
- **Lines**: ~75 lines per file
- **Features**:
  - Linear joint interpolation
  - Minimal IK (2 calls vs 30 for MoveL)
  - Blue trajectory visualization
  - Curved Cartesian path
  - 15x faster than MoveL for repositioning

### Performance Metrics
| Metric | MoveJ | MoveL |
|--------|-------|-------|
| IK Calls | 2 | 30 |
| Computation Time | ~0.1s | ~1.5s |
| Path Shape | Curved | Straight |
| Trajectory Color | Blue | Red |
| Use Case | Repositioning | Drawing/Welding |
